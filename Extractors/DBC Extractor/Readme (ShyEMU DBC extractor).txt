---This will extract the DBC's required to run ShyEMU---
--------------------------------------------------------

-------------------------------------------------------
--- MPQE.exe - SFmpq.dll - ShyEMU DBC extractor.bat ---
-------------------------------------------------------

Step 1) You will need to copy the 3 files above into your clients data 
locale directory (look below)


--------
--enUS--
--------
Windows 7/Windows Vista: C:\Users\Public\Games\World of Warcraft\Data\enUS

Windows xp: C:\Program Files\World of Warcraft\Data\enUS

--------
--enGB--
--------
Windows 7/Windows Vista: C:\Users\Public\Games\World of Warcraft\Data\enGB

Windows xp: C:\Program Files\World of Warcraft\Data\enGB


Step 2) Run ShyEMU DBC extractor.bat and let it run (it should take about 5 - 10 mins)

Step 3) Once it is done you will see a folder called "DBC"

Step 4) Copy it to your ShyEMU server directory


--You now have the latest DBC's for your server--

NOTE: You will only need to do this every time a new patch is released.
